# Hermes Pipeline

## Project Overview
Hermes Pipeline is an engineering portfolio project demonstrating end-to-end proficiency across DevSecOps, platform engineering, SRE, and full-stack development. It serves as a production-grade reference implementation showcasing:

- **DevSecOps**: CI/CD pipelines with integrated security scanning, secrets detection, SAST/DAST, and policy-as-code
- **Platform Engineering**: Infrastructure-as-code, developer self-service tooling, and internal platform abstractions
- **SRE**: Observability (metrics, logs, traces), SLOs/SLAs, alerting, incident response runbooks, and reliability patterns
- **Full Stack**: End-to-end application development spanning frontend, backend, APIs, and data layers
